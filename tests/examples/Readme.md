## Introduction

Folder contains example test files for the playwright framework.These are not the real test for application. New user can refer these files to understand the framework.

**_example.spec.ts_** - Contains the example test file.  
**_fixture-apppage-wrapper.v4.spec.ts_** - Test uses the AppPage wrapper which is provided from fixture.  
**_fixture-newpage.spec.ts_** - Example test for having two different browser context.This concept can be used for multiple logins.  
**_navigation-apppage-wrapper.v4.spec.ts_** -Test uses the AppPage wrapper without fixture.  
**_teststep-example.v4.spec.ts_** - Example for playwright test.step function.
